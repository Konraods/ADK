# Edycja obrazów

Aplikacja służąca do edycji obrazów medycznych.

# Instalacja modułów

Użyj [pip](https://pip.pypa.io/en/stable/) żeby zainstalować:

```bash
pip install easy_gui
```
```bash
pip install opencv-python
```
